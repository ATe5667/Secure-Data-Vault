# Secure-Data-Vault
A desktop application made using Python to securely store personalised data.

The application is currently under development.


## Excepted Features:
1) Notes
2) Passwords
3) Password generator
4) User Accounts
5) Auto-lock
6) Themes
7) Tutorial
8) Reminders/Calendar
9) Recycle bin
10) Log
11) Chat
12) Panic mode
